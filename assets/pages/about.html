Geometrik : Le jeu qui défie vos capacités comme joueur

Geometrik est un jeu d'action inspiré du jeu populaire "Vampire Survivors", où vous contrôlez des héros géométriques dans un monde envahi par des formes ennemies. Choisissez votre champion - un guerrier ou un tireur - et affrontez des vagues d'adversaires de plus en plus difficiles.

Dans cet univers minimaliste mais captivant, chaque entité est représentée par des formes géométriques distinctives. Les héros, ennemis et boss possèdent des capacités uniques liées à leur forme, créant une expérience de jeu à la fois simple visuellement et stratégiquement profonde.

Collectez des améliorations, développez votre personnage et tentez de survivre le plus longtemps possible face à l'assaut implacable des formes géométriques hostiles. Affrontez des boss impressionnants et découvrez jusqu'où vos compétences peuvent vous mener!

Caractéristiques principales :

  • Un Client Web et un Client Desktop!
  • Système de score et classement des meilleurs joueurs
  • Système de score pour prouver votre performance!
  • Des ennemies pour vous défier et vous mettre a l'épreuve!
  • Des améliorations personnalisées pour vous offrir une experience unique!
  • Ennemis et boss variés représentés par différentes formes géométriques
  • Mode survie avec difficulté croissante

Technologies utilisées

Front-end

  • HTML5
  • CSS3
  • JavaScript
  • C# pour le client desktop
  • Canvas API

Back-end

  • PHP
  • MySQL

  • Git & GitHub
  • Visual Studio Code
  • Figma pour le design

Bibliothèques

  • Pixi.js pour créer le jeu
